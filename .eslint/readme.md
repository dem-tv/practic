# Правила для Eslint Flat Config (Eslint 9.20+)

В качестве основы используются правила из open source пакетов

* [eslint-config-airbnb-base](https://github.com/airbnb/javascript/tree/master/packages/eslint-config-airbnb-base/rules)

* [eslint-config-airbnb-typescript](https://github.com/iamturns/eslint-config-airbnb-typescript/tree/master/lib)

Правила сохранены локально, т.к. указанные пакеты не были обновлены под flat config.

В качестве основы использованы правила `eslint-config-airbnb-base`, откуда были удалены некоторые устаревшие правила и правила, которые есть в prettier, а так же некоторые правила заменены на аналогичные для `typescript`, по шаблону из пакета `eslint-config-airbnb-typescript`.
